¿Qué es PKCS#12?

PKCS#12 es un formato de archivo que contiene tanto el certificado público como la llave privada en un solo archivo protegido por contraseña. Este formato es ampliamente compatible con navegadores, sistemas operativos y aplicaciones.

¿Cómo usar este generador?

  1. Selecciona tu archivo de certificado (.cer)
  2. Selecciona tu archivo de llave privada (.key)
  3. Ingresa una contraseña segura para proteger el archivo .p12
  4. Haz clic en "Generar archivo .p12"
  5. El archivo se descargará automáticamente
Nota de seguridad: Todo el proceso de conversión se realiza completamente en tu navegador usando JavaScript. Tus archivos y contraseñas nunca salen de tu computadora.
