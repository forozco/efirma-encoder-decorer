¿Cómo usar este decodificador?

  1. Pega el string base64 del archivo PKCS#12 (.p12) en el área de texto
  2. Ingresa la contraseña con la que fue protegido el archivo
  3. Haz clic en "Decodificar PKCS#12"
  4. Visualiza toda la información del certificado y llave privada
Nota de seguridad: Todo el proceso de decodificación se realiza completamente en tu navegador usando JavaScript. Tu string base64 y contraseña nunca salen de tu computadora.
